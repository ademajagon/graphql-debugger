3. `graphql-debugger` - CLI for GraphQL Debugger

## Getting started

### Prerequisites

1. [Node.js](https://nodejs.org/en/) (v18 or higher)
2. [PNPM](https://pnpm.io/) (v6 or higher)

### Installation

```bash
pnpm install
```

### Testing

```
pnpm test
```
